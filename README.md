# my-portfolio

## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm start
```

## Deploy
Via GitHub Pages. [link](https://create-react-app.dev/docs/deployment/#github-pages "CRA Github")

```
$ npm run build
$ gh-pages -d build
```
